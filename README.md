# Data-Analytics-Visualization-Project

1. Install Requirements

Download the project folder and install all required packages:

pip install -r requirements.txt

2. Run Cleaning Scripts

Before launching the dashboard, run the cleaning script to prepare the data:

python clean_average_age.py


(This creates the cleaned age dataset used in the app.)

3. Launch the Dashboard

Start the Dash application:

python dash_app.py


Then open the link shown in your terminal (e.g. http://127.0.0.1:8050/) to view the dashboard.
